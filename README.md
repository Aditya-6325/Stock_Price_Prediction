# Stock_Price_Prediction
Here we will be exploring how to predict the next day’s stock closing price using historical data.   The goal is not to perfectly forecast the market but to build a simple machine learning model to understand price trends.

Features : 
- It will fetch the historical stock prices with `yfinance`
- Then it will preprocess the data and create features
- Then we train a Linear Regression model on this data
- Then evaluate the model performance
- Finally we predict future stock prices

This program has been executed in Jupyter Notebook.

How to Run this: 

1. Install required libraries:
   Numpy
   Pandas
   Matplotlib
   yfinance
   Scikit-learn
   Joblib

   run following command :
     pip install numpy pandas matplotlib yfinance scikit-learn joblib

2. Open `Stock_Price_Prediction.ipynb` in Jupyter Notebook
   
3. Run all cells from top to bottom
